qi.js

Manage your image sources directly in html code.

 Download  Github

Introduction

Queried Images is simple in use, lightweight and quite flexible javascript module, that automatically changes image sources in your html code depending on browser's window width.

It allows you to override default media queries included in module or even create your own breakpoints groups. And the best part is that the whole minified file has only 781 bytes of code!

Qi.js can be a nice alternative for hiding images via css classes and can help you to organize your code on html layer.

If you have some doubts about how it works, feel free to try out emulation on the right.

Basic usage

<html>
<head>
    <script src="qi.js"></script>
</head>
<body>
    <img qi-small  = "http://via.placeholder.com/1366x768" 
         qi-medium = "http://via.placeholder.com/1600x1900" 
         qi-large  = "http://via.placeholder.com/1920x1080" />
    <script>
        new Qi();
    </script>
</body>
</html>

To start using qi.js on basic, there are only three easy steps that you must follow:

  1. pull downloaded module to your html,
  2. add dynamic qi sources to every <img> tag that you like,
  3. before closing </body> tag initialize new Qi object (or create event that fires after document is loaded).

Default media queries

By default, Qi object has three media breakpoints based on device's width:

  • qi-small - for 0 pixels or more,
  • qi-medium - for 640 pixels or more,
  • qi-large - for 1024 pixels or more.

The important thing is that larger queries overrides smaller, like it is in a Zurb's Foundation approach.

Custom and multiple breakpoints

As mentioned, there is an option to invoke personalized breakpoints or override default one. To do that, just pass JSON object when creating Qi object like shown on the snippet.

In the same way you can build your own breakpoints names at your discretion!

You can create as many Qi objects as you want, but pay attention to given names. If only one of them is repeated, it can cause unwanted assignments.

<-- creating new breakpoints à la Bootstrap --> 
<img qi-xs = "http://via.placeholder.com/100x100" 
     qi-sm = "http://via.placeholder.com/200x100" 
     qi-md = "http://via.placeholder.com/300x100" 
     qi-lg = "http://via.placeholder.com/400x100" />

<script>
    new Qi({
        xs: 0,
        sm: 768,
        md: 992,
        lg: 1200
    });
</script>
